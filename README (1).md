# NSYGHT — SEO & AI Search Agency

Website for NSYGHT digital marketing agency, built with pure HTML/CSS/JS and hosted on Netlify.

## Structure

```
nsyght/
├── index.html                    ← Homepage
├── contact.html                  ← Contact / Lead form
├── netlify.toml                  ← Netlify config + redirects
├── robots.txt                    ← SEO crawler rules
├── sitemap.xml                   ← XML sitemap
├── images/
│   ├── logo.png                  ← NSYGHT_3.png (nav - white bg)
│   └── logo-white.png            ← NSYGHT_Logo-white-text (footer)
└── README.md
```
